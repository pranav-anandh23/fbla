# FBLA Connect

Created by: Mitul Ajith / Saanvi Akurati / Pranav Anandh

Purpose: A mobile app prototype representing an official app for FBLA members, to be used in the Mobile Application Development event. 


## Description

This app aims to model what an official app for FBLA members could look like. This means getting the info out to members at a quick glance, without having to scour online or deal with other apps, creating a singular space for everything FBLA. This involves displaying the latest news, stats, resources for each FBLA event, and any events you might want to add on your calendar, such as a competition or a study date. By accomplishing this, FBLA members can continue to keep up to date with their needs any time, from any place.


### Technical Summary

Platform: Android
Language: Java
IDE: Android Studio
Dependencies: AndroidX defaults, Material Components for TextInput (com.google.android.material:material:1.12.0)


### Resource Attribution

Icons: Default Android Studio icons

External Content: All FBLA rubrics, images, and videos included are official FBLA resources downloaded from FBLA.org or official FBLA materials.

Materials are used for educational use and representation of official content.


### How to Build/Run

1. Open in Android Studio

2. Sync Gradle

3. Run on emulator or device. Layout works best on a phone.
